---

## 📑 Repository Overview
This hub is designed to take a learner from "Hello World" in Nginx to managing enterprise-level traffic distribution.

### 1. 📂 [Projects](./project)
- **Two-Tier Architecture:** Full-stack integration with Node.js & MySQL.
- **SSL Termination:** Secure communication using OpenSSL and Nginx.
- **Reverse Proxying:** Masking backend ports for enhanced security.

### 2. ⚖️ [Load Balancer](./nginx-as-LoadBalancer)
- **High Availability:** Distributing traffic across 3 Node.js backend instances.
- **Algorithms:** Implementation of Round-Robin and Weighted Load Balancing.
- **Upstream Management:** Grouping server clusters for horizontal scaling.

### 3. 🌐 [Web Server](./nginx-as-WebServer)
- **Virtual Hosting:** Hosting multiple sites (`site1`, `site2`, `site3`) on a single IP.
- **Server Blocks:** Domain-based routing and isolated web roots.

### 4. 📝 [Deep-Dive Notes](./notes)
- Detailed documentation on Nginx architecture, configuration syntax, and troubleshooting commands.

---

## 🛠️ Core Skills Demonstrated
- **Proxy Management:** `proxy_pass`, `proxy_set_header`, and `upstream` logic.
- **Security:** SSL/TLS setup, `.key` and `.crt` management, and Firewall rules.
- **Virtual Hosting:** Managing multiple `.conf` files via `conf.d`.
- **Backend Integration:** Connecting Nginx with Node.js/Express and MySQL databases.

## 🚀 How to Navigate
Each sub-folder contains its own specific `README.md` with detailed installation steps.
- To learn **Virtual Hosting**, start with the [Web Server](./nginx-as-WebServer) folder.
- To learn **Scaling**, move to the [Load Balancer](./nginx-as-LoadBalancer) folder.
- To see a **Full Project**, check the [Main Project](./project) folder.
